# Prodesk Cloud – Corporate Brand Landing Page

A responsive corporate landing page built using HTML, CSS and JavaScript. The project demonstrates responsive layouts, reusable components, dark mode support and interactive UI elements without using any front-end frameworks.

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)

## Features

* Responsive navigation
* Hero section
* Features and Services sections
* Statistics section
* Testimonials
* Pricing cards
* FAQ accordion
* Contact form
* Dark/Light theme toggle
* Mobile navigation
* Automatic footer year

## Project Structure

```
PRODESK/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Running the Project

1. Clone the repository.

```bash
git clone https://github.com/kriti-2006/corporate-brand-landing.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

## Responsive Design

The landing page is designed to work across desktop, tablet and mobile devices using CSS Grid, Flexbox and media queries.

## JavaScript Functionality

The JavaScript handles:

* Mobile navigation
* Theme switching with localStorage
* FAQ accordion
* Dynamic footer year
